API Rest Node.JS

Développement d'une API REST en Node.js avec Express, utilisant Sequelize pour la gestion de la base de données MySQL.

API REST : Opérations CRUD implémentées avec Express. Intégration de la base de données : Connecté à une base de données SQL en utilisant l'ORM Sequelize. Chiffrement des mots de passe des utilisateurs : Implémenté avec Bcrypt. Gestion des erreurs : Gère les erreurs avec les codes 404, 401 et 500. Jeton JWT : Utilisé pour des communications sécurisées.

Pierre Feuille Ciseaux

Lors de la réalisation de ce projet, j'ai développé un jeu "Pierre-Feuille-Ciseaux" en utilisant HTML, CSS et JavaScript vanilla. L'objectif principal de ce projet était d'apprendre les bases de JavaScript et de perfectionner mes compétences en manipulation du DOM. J'ai conçu une interface utilisateur interactive et dynamique, permettant aux joueurs de sélectionner leurs coups et de voir instantanément les résultats, le tout avec un design responsive et convivial.

Ce projet m'a permis de renforcer mes compétences en développement front-end et de comprendre les interactions fondamentales entre le code JavaScript et le DOM.
